Further Information

  • The code running the model fitting is in the function/script simulate_usanalysis.R. This function does the sampling, repeatedly calls the underlying simulation model encoded in simulate_basicbacteri.R, and records the results from each simulation. Check the help file for both functions to see how to use them.
  • Good papers explaining uncertainty and sensitivity analysis in a bit more detail are (Hoare, Regan, and Wilson 2008; Marino et al. 2008).
