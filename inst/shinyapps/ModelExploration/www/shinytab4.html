Further Information

  • For this app, the underlying function running the loop over different parameter values is called simulate_modelexploration.R. This function in turn calls the bacteria model simulator simulate_basicbacteria.R.
  • You could for instance write your own function with that loops over some parameter and for each parameter calls one of the model simulator, e.g. simulate_basicbacteria.R or simulate_basicvirus.R or simulate_virusandIR.R. See the package vignette for an example.
  • You can also call the simulate_modelexploration.R function directly, and as needed embed it in further R code.
  • You could also modify the simulate_modelexploration.R, e.g. such that it uses the virus model for model exploration.
  • Some ‘real world’ examples where models have been used to explore outcomes of interest as function of parameters for simple models applied to TB, see e.g. (Antia, Koella, and Perrot 1996) (specifically Figure 2B) or Malaria, see e.g. [@].
